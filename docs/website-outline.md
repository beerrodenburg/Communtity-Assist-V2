# Community Assist — Landing Page Outline (Stitch-ready)

A single long-scroll, donate-first landing page. Narrative arc: **impact-led funnel**. Each section below is a self-contained brief you can paste into **Google Stitch** together with the listed `pages/slide-XX.jpg` reference(s) to generate a mockup.

---

## How to use this with Google Stitch

1. Copy the **Global style preamble** (below) once.
2. For each section: paste `Preamble + the section's PROMPT block`, and **attach the slide JPG(s)** named in "Attach."
3. Generate, refine, move to the next section. Build order = top to bottom.
4. The metadata around each PROMPT (Goal / Attach / Notes) is for us — you mainly paste the **PROMPT** block.

### Global style preamble (prepend to every section prompt)
> Design one section of a modern, fundraising-focused charity landing page for **Community Assist**, a Bali-based non-profit collective that runs community events to fund children's school scholarships. Overall style: **clean, editorial, lots of whitespace — premium, not cluttered.** Brand colors: sky blue `#69A0C9` and deep blue `#4661AD` (primary actions, buttons, links), warm off-white background `#F1EFF1`, white cards, with orchid `#D4A5CC`, hot pink `#E94E9C` and yellow `#F4C84B` as sparing accents. Headings in a bold geometric sans (like General Sans); body in a clean sans (like Inter). Rounded corners (~20px) and pill buttons. The brand icon is a **horizontal-striped "scanline" heart**. Use occasional small **hand-drawn doodle accents** (stars, hearts, underlines) in blue — sparingly. Photography is candid, vibrant, film-style shots of Bali community life. Fully responsive, show the **desktop** layout. The single most prominent button anywhere is a deep-blue **"Donate"** button.
